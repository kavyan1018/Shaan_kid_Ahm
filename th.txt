what is html ?
--> html is a markup language used for creating web pages. HTML uses a system of tags and attributes to define elements such as headings, paragraphs, links, images, and more. 

html --> HyperText Markup Language

whta is static website ?
--> A static website is a type of website that displays the same content for every user. Static websites are typically built using HTML and CSS, and they do not require server-side processing or databases to deliver content.

what is dynamic website ?
--> A dynamic website is a type of website that displays different content and allows user interaction by using client-side or server-side scripting. 


client-side scripting is executed on the user's browser

server-side scripting is executed on the server before the page is sent to the user's browser
